Part 1: Arrays

Storing lists of things in JavaScript

What is an Array?

An array is a special variable that can hold more than one value at a time.

Imagine you have a backpack. Instead of carrying one pencil, you can carry a whole pencil case with 10 pencils inside. An array is like that pencil case — one container, many items inside!

Creating an Array

Example — Making an array of fruits
// Two ways to create an array:

// Way 1: Use square brackets (most common!)
let fruits = ["Apple", "Banana", "Cherry"];

// Way 2: Use the Array constructor
let colors = new Array("Red", "Blue", "Green");

Accessing Array Elements

Each item in an array has a position number called an index. The first item is at index 0 (not 1!).

Think of a row of lockers numbered starting from 0. Locker 0 has the first item, Locker 1 has the second, and so on.
Example — Getting items by their index
let animals = ["Cat", "Dog", "Fish", "Bird"];

// Index:    0       1       2       3

animals[0]   // "Cat"   (first item)
animals[1]   // "Dog"   (second item)
animals[2]   // "Fish"  (third item)
animals[3]   // "Bird"  (fourth item)

Changing & Adding Items

Example — Changing and adding
let toys = ["Lego", "Puzzle", "Ball"];

// Change the second item (index 1)
toys[1] = "Robot";

// Add a new item at the end (index 3)
toys[3] = "Doll";

Array Length

Use .length to find out how many items are in your array.

Example — Counting items
let snacks = ["Chips", "Cookies", "Candy", "Popcorn"];

snacks.length   // 4 items in the array!

Array Methods: Push, Pop, Shift, Unshift

These are special commands that let you add or remove items from an array.

Imagine a line of people waiting for ice cream. Push = someone joins at the back. Pop = the last person leaves. Unshift = someone cuts to the front. Shift = the first person leaves.

push() — Add to the END

Example — push
let queue = ["Alice", "Bob"];
queue.push("Charlie");
// queue is now: ["Alice", "Bob", "Charlie"]

pop() — Remove from the END

Example — pop
let stack = ["Plate 1", "Plate 2", "Plate 3"];
let removed = stack.pop();
// removed = "Plate 3"
// stack is now: ["Plate 1", "Plate 2"]

unshift() — Add to the BEGINNING

Example — unshift
let line = ["Bob", "Charlie"];
line.unshift("Alice");
// line is now: ["Alice", "Bob", "Charlie"]

shift() — Remove from the BEGINNING

Example — shift
let racers = ["Flash", "Speedy", "Turbo"];
let first = racers.shift();
// first = "Flash"
// racers is now: ["Speedy", "Turbo"]

push & pop (Fast!)

Work at the end of the array. No need to move other items around, so they're super quick.

unshift & shift (Slower)

Work at the beginning of the array. All other items need to shuffle over, so they take more time.

Part 2: Conditional Statements

Making decisions in your code

The if Statement

An if statement lets your code make a decision. "If something is true, do this."

It's like asking: "Is it raining?" If YES, grab an umbrella. If NO, wear sunglasses. Your code can make the same kind of choices!

Simple if

Example — Simple if
let temperature = 35;

if (temperature > 30) {
    document.write("It's a hot day! Stay cool!");
}

if ... else

What if the condition is false? Use else to handle that!

Example — if / else
let age = 10;

if (age >= 18) {
    // This runs if age is 18 or more
    document.write("You can drive a car!");
} else {
    // This runs if age is less than 18
    document.write("You're too young to drive, but you can ride a bike!");
}

if ... else if ... else

Sometimes there are more than two choices. Use else if to check more conditions!

Example — Multiple conditions
let score = 85;

if (score >= 90) {
    document.write("Amazing! You got an A!");
} else if (score >= 70) {
    document.write("Great job! You got a B!");
} else if (score >= 50) {
    document.write("Not bad! You got a C. Keep practicing!");
} else {
    document.write("Oops! Let's study more and try again!");
}

Boolean Values — Truthy & Falsy

JavaScript treats some values as "truthy" (like true) and some as "falsy" (like false).

Example — Truthy and falsy values
// These are ALL "falsy" (treated like false):
//   0, "" (empty text), null, undefined, NaN

let cookies = 0;

if (cookies) {
    document.write("You have cookies!");
} else {
    document.write("No cookies left! 0 is falsy.");
}

The switch Statement

A switch is like a menu of choices. It checks one value and picks the matching option.

It's like a vending machine. You press button A, B, or C, and you get a different snack depending on which button you pressed!
Example — What day is it?
let day = "Monday";

switch (day) {
    case "Monday":
        document.write("Start of the week! Let's go!");
        break;
    case "Friday":
        document.write("Almost weekend! Woohoo!");
        break;
    case "Saturday":
    case "Sunday":
        document.write("Weekend! Time to play!");
        break;
    default:
        document.write("Just a regular school day.");
}

What Happens Without break?

Without break, the code "falls through" and keeps running the next cases too!

Example — Forgetting break (oops!)
let fruit = "Banana";

switch (fruit) {
    case "Apple":
        document.write("Apple picked!\n");
    case "Banana":
        document.write("Banana picked!\n");
        // No break here! Falls through...
    case "Cherry":
        document.write("Cherry picked too!\n");
        // Also no break! Keeps going...
    default:
        document.write("Default also runs!");
}

Part 3: Loops

Doing things over and over again

The while Loop

A while loop keeps running as long as a condition is true. It checks the condition before each run.

Imagine eating candy from a jar. "While there is candy in the jar, take one and eat it." When the jar is empty, you stop!

How it works step by step:

  1. Check the condition. Is it true?
  2. If YES, run the code inside the loop.
  3. Go back to step 1 and check again.
  4. If NO, stop the loop and move on.
Example — Counting from 1 to 5
let count = 1;

while (count <= 5) {
    document.write("Count is: " + count + "\n");
    count++;   // Add 1 to count (very important!)
}
Example — Summing numbers
let n = 0;
let total = 0;

while (n < 3) {
    n++;
    total += n;   // Same as: total = total + n
    document.write("n = " + n + ", total = " + total + "\n");
}

The do...while Loop

This loop is special: it always runs at least once, then checks if it should keep going.

It's like tasting a new food. You ALWAYS take at least one bite first, THEN decide if you want more!
Example — do...while runs at least once
let num = 10;

do {
    document.write("num is: " + num + "\n");
    num++;
} while (num < 5);

document.write("Loop ended! num was already 10, which is NOT < 5.\n");
document.write("But it still ran ONCE because it checks AFTER running.");
Example — Countdown with do...while
let rocket = 5;

do {
    document.write(rocket + "...\n");
    rocket--;   // Subtract 1
} while (rocket > 0);

document.write("BLAST OFF!");

The for Loop

The for loop is the most popular loop. It puts everything in one neat line: where to start, when to stop, and how to count.

It's like a running track. You know where the start line is, where the finish line is, and how big each step is. "Start at lap 1, stop at lap 5, add 1 lap each time."

The 3 Parts of a for Loop:

The anatomy of a for loop
//  for (START;     CONDITION;   STEP)
    for (let i = 0;   i < 5;       i++) {
        // START:     let i = 0   (begin at 0)
        // CONDITION: i < 5       (keep going while i is less than 5)
        // STEP:      i++         (add 1 after each loop)
    }
Example — Counting 0 to 4
for (let i = 0; i < 5; i++) {
    document.write("i = " + i + "\n");
}

Looping Through an Array

This is one of the most common things you'll do with a for loop!

Example — Printing every item in an array
let pets = ["Hamster", "Goldfish", "Parrot", "Kitten"];

for (let i = 0; i < pets.length; i++) {
    document.write("Pet #" + (i + 1) + ": " + pets[i] + "\n");
}
Example — Adding up prices
let prices = [5, 10, 3, 7];
let total = 0;

for (let i = 0; i < prices.length; i++) {
    total += prices[i];
    document.write("Added $" + prices[i] + " -> Total: $" + total + "\n");
}

document.write("Grand total: $" + total);

break & continue

These are special commands that give you extra control over loops.

break — Stop the loop completely

You're searching your backpack for your phone. Once you find it, you STOP searching. That's break!
Example — Finding a number
let numbers = [4, 8, 15, 16, 23, 42];

for (let i = 0; i < numbers.length; i++) {
    document.write("Checking: " + numbers[i] + "\n");
    if (numbers[i] === 16) {
        document.write("Found 16! Stopping the search.");
        break;   // EXIT the loop right now!
    }
}

continue — Skip this one, go to the next

You're picking apples from a tree. If you see a rotten one, you skip it and move on to the next apple. That's continue!
Example — Print only odd numbers
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue;   // Skip even numbers
    }
    document.write(i + " is odd!\n");
}

Quick Reference

Everything at a glance

Arrays

Lists of items stored in one variable. Access by index starting from 0. Use push/pop for fast adding/removing at the end.

if / else

Make decisions: "If this is true, do A, otherwise do B." Chain with else if for more choices.

switch

Like a menu of choices. Matches a value to a case. Don't forget break or it falls through!

while

Repeats while a condition is true. Checks before each loop. Could run zero times.

do...while

Like while, but checks after running. Always runs at least once.

for

The most common loop. Has 3 parts: start, condition, step. Great for looping through arrays.
