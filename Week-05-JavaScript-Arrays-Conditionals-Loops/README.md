## üìù Key Takeaways

- **Arrays** store multiple values in a single variable, accessed by index starting at 0
- **if/else** lets your code make decisions based on conditions
- **switch** compares one value against many possible cases
- **while** loops repeat while a condition is true
- **do...while** always executes at least once
- **for** loops are ideal for counting and iterating through arrays
- **break** exits a loop early; **continue** skips to the next iteration
